Agendamento de Quadras - Backend

API Java Web manual com Servlets, JDBC, Maven, GlassFish e PostgreSQL.

Rotas principais

  • GET /api/usuarios - lista usuarios
  • POST /api/usuarios - cadastra usuario
  • POST /api/login - autentica usuario
  • GET /api/quadras - lista quadras
  • POST /api/quadras - cadastra quadra
  • GET /api/agendamentos - lista agendamentos
  • POST /api/agendamentos - cria agendamento
  • GET /api/agendamentos/disponiveis?quadraId=1&data=2026-06-01 - lista horarios livres
